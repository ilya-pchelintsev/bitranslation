train_transformer.sh - скрипт для обучения трансформера. Устанавливает все нужные библиотеки (может быть почти все), умеет собирать датасет и запускать тензорборд. Параметры:
  --base-dir - базовая диретория, куда будут складываться все данные (по умолчанию /data/$USER/bidirectional_transformer)
  --source-lang - язык, с которого переводим (en, fr, по умолчанию en)
  --target-lang - язык, на который переводим (en, fr, по умолчанию fr)
  --install-lib - флаг того, что нужно установить библиотеки (по умолчанию false)
  --prepare-dataset - флаг того, что нужно собрать датасет для конкретной пары языков (по умолчанию false)
  --run-tensorboard - флаг того, что нужно запустить тензорборд (по умолчанию false)

prepare_dataset.sh - скрипт для сборки датасета, используется предыдущим скриптом, поэтому его надо положить в ту же директорию. Параметры:
  --base-dir - базовая диретория, куда будут складываться все данные (по умолчанию /data/$USER/bidirectional_transformer)
  --source-lang - язык, с которого переводим (en, de, по умолчанию en)
  --target-lang - язык, на который переводим (en, de, по умолчанию de)
  --install-lib - флаг того, что нужно установить библиотеки (по умолчанию false)

run_jupyter.sh - устанавливает python окружение и запускает jupyter. Параметры:
  --environments-dir - директория с окружениями (по умолчанию /data/$USER/environments)
  --python - название окружения и одновременно версия питона (по умолчанию python3)
  --ip - IP адрес машины, на которой запускается (наверное можно автоматизировать, но пока по умолчанию указан 95.213.170.234 - IP кластера ШАДа)
  --port - порт (по умолчанию 25400)
